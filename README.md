# 🎮 Dự án Web Game (Y8 Clone) - Nền tảng Mini-game T36

> Một nền tảng tổng hợp các mini-game kinh điển được thiết kế lại với giao diện Neon hiện đại. Dự án được phát triển bởi nhóm 3 thành viên.

**🌐 Chơi game trực tiếp (Live Demo):** https://thobuiquang.github.io/mini-game-platform/frontend/

## 🛠 Công nghệ sử dụng
- **Frontend:** HTML5, CSS3, JavaScript, AI-Assisted Generation
- **Backend:** Python (Flask/FastAPI/Django...)
- **Design & Quản lý:** Figma, GitHub, Google Docs/Sheets

## 📁 Cấu trúc dự án
- `/frontend`: Chứa mã nguồn giao diện web (được chuyển từ Figma sang code bằng AI).
- `/backend`: Chứa logic hệ thống, API xử lý bằng Python.
- `/docs`: Tài liệu đặc tả hệ thống (SRS)
